# 🐺 Projeto Lobo Tecnológico - Programa Transforme-se

Este repositório contém o código de uma Landing Page moderna e responsiva desenvolvida para o projeto Lobo Tecnológico, criado sob a iniciativa do programa Transforme-se — uma parceria entre o Instituto PROA e a Serasa Experian.

🛠️ Tecnologias Utilizadas:

    HTML5: Estruturação semântica da página.

    CSS3: Estilização, layout responsivo (Flexbox) e identidade visual baseada no protótipo.

    Google Fonts: Integração da tipografia
